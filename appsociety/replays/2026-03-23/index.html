3. Discussions clés

Web-to-App (Web2App) — Le grand sujet du soir

  • Concept : onboarding + paiement sur le web, pas dans l'app. Skip la commission Apple (15-30%).
  • Avantages : attribution parfaite (pixel TikTok direct, pas besoin de MMP), pas d'installation requise, personnalisation par pub (passer l'ID de la pub dans l'URL → l'onboarding rebondit dessus).
  • Paiement : Apple Pay / Google Pay fonctionnent sur le web = friction minimale.
  • Outils : RevenueCat "Funnel" (templates d'onboarding web), Adapty (concurrent, gratuit < 5K€/mois), Paddle pour la TVA.
  • Risque Apple : Capacitor/web views peuvent être détectés. Astuce : démarrer en app native, puis rediriger vers le web via une config serveur (OTA update).
  • Podcast Rise Science dans le rapport RevenueCat : onboarding web = paradigme différent, bien détourer le problème avant le paywall.
  • Consensus : c'est peut-être le "move de 2026". À tester.

Hard paywall vs Soft paywall vs Freemium

  • Harry clarifie les termes : hard = obligé de s'engager, soft = peut fermer et accéder en gratuit, freemium = accès direct avec limites.
  • Rapport RevenueCat : hard paywall convertit 4-5x mieux. L'écart se rattrape au bout d'un an en freemium.
  • Noah : "L'avantage du hard, c'est que tu gagnes tellement plus tôt que tu peux avancer plus vite."
  • Harry : soft paywall. Envisage de tester le hard pendant une semaine.
  • Adilbh : soft paywall avec free trial 7 jours.
  • Conseil général : A/B testing, et toujours tester un prix plus élevé.

Google Stitch et le "Vibe Design"

  • Valentin teste Google Stitch (mise à jour récente) : résultat initial décevant, mais l'édition inline est prometteuse.
  • Syx : l'utilise pour des dashboards → envoie à Claude Opus → résultats satisfaisants.
  • KévinM : sa boîte a abandonné Figma, full vibe design avec un design system (Chakra) + code → reverse vers un outil design.
  • Figma Make : même concept, choix entre Claude et Gemini.
  • Consensus : le "vibe design" est aux débuts, comme le vibe coding il y a un an.

TikTok et l'authenticité

  • Romain : posts ultra-minimalistes (photos de bureau + texte) = meilleur engagement que du contenu travaillé.
  • Harry raconte l'histoire d'Anna : carousel fait à l'arrache → 40K vues. "Le mode YOLO est le plus efficace sur TikTok."
  • Valentin : exemple des faux portefeuilles artisanaux (IA + émotion fabriquée = millions de vues). L'émotion > la production.
  • TikTok = pas de règle fixe. Dès qu'une "méta" émerge, elle est annulée par l'algo.
  • Copier les concurrents d'autres pays = stratégie valide.

  • Romain : a essayé le canal coworking normal, impossible de se concentrer (sujets trop intéressants).
